User Experience

Deze opdracht gaat over het presenteren van een project.
Momenteel houd ik mij bezig met een project dat ik moet presenteren.
Dit doe ik met andere studiegenoten.
Om het project goed te kunnen presenteren,
heb ik een scenario geschreven, een storyboard en een 3D prototype gemaakt.
Het project is om een actief en leerzaam spel te maken voor kinderen.

Reflectie

Door het maken van deze opdracht heb ik geleerd om mezelf in de doelgroep in te beelden.
